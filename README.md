# Command Hub

Ryan's personal command center — stocks, weather, Gmail digest, YouTube queue, shopping list, and idea capture in one dashboard.

**Live dashboard:** https://ryry84.github.io/command-hub

## Setup
1. Enable GitHub Pages: Settings → Pages → Source: Deploy from branch → Branch: main → Save
2. GitHub Actions will auto-update stock prices every 15 min during market hours
3. Open the dashboard and use the ⚙ Config section at the bottom to set your commute addresses
4. Use ⚙ Positions in the Portfolio card to enter your share counts
